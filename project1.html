1. Crop Disease and Insect Detection Module (Jetson Nano with Flask Dashboard)

Working Description

This module is powered by the NVIDIA Jetson Nano, a compact edge AI device capable of running deep learning models locally for real-time inference.

It is connected to a Raspberry Pi 8MP camera module that captures high-resolution images of crops.

A custom-trained YOLOv8 object detection model, optimized with TensorRT for Jetson Nano, detects 33 different classes of crop diseases and insect pests affecting crops like Blackgram, Cotton, and Soybean.

The model processes images locally on the Jetson Nano without requiring internet connectivity, enabling usage in remote farm areas.

Detection results (disease/insect type, confidence score, timestamp, and optional geolocation) are logged in CSV format for export and analysis.

The Jetson Nano runs a Flask-based web server that hosts a user-friendly dashboard accessible from any device on the local network.

The Flask web interface displays live camera feeds with detection bounding boxes and labels in real time.

It also provides functionalities to start/stop the inference, view recent detection data, and export logs in CSV/Excel format.

Cooling fans keep the Jetson Nano temperature optimized during continuous operation.

A USB WiFi adapter optionally facilitates local network connectivity for remote monitoring on smartphones or laptops.

  • The Jetson Nano powers up and initializes all software modules and camera.
  • The Raspberry Pi camera captures the live video feed of the crops.
  • Images are processed through the optimized YOLOv8 model running locally.
  • Detected diseases and pest instances are highlighted on the live video feed shown on the Flask dashboard.

  • Users can access the Flask web interface on devices connected to the same network, allowing easy real-time monitoring.
  • The CSV logs can be exported for offline analysis to track trends and outbreaks over time.

2. Animal Intrusion Detection Module (ESP32-CAM with LoRa and Flask Dashboard)

Working Description

An ESP32-CAM microcontroller equipped with an OV5640 5MP camera sensor captures images to detect wild animals in farm or forest areas.

A TinyML animal detection model trained on Edge Impulse runs locally on the ESP32-CAM to identify animals such as tigers, leopards, bears, and elephants.

On detection, the ESP32-CAM sends a unique animal code wirelessly using LoRa (Long Range) communication to a nearby ESP32 receiver microcontroller.

The receiver ESP32 collects detections and relays them to a Flask web server hosting a real-time animal intrusion dashboard.

The Flask web interface displays live detection alerts, showing animal type, timestamp, and location details.

The dashboard supports historical logging and can trigger audible or visual alerts to immediately inform the farmer.

This system operates independently of WiFi or cellular networks, making it ideal for remote agricultural fields.

Workflow Summary

  • ESP32-CAM continuously captures images and performs local animal detection.
  • Detected animal type is transmitted as a code over LoRa to the receiver ESP32.
  • Receiver ESP32 sends received data to the Flask server on the local network.
  • The Flask dashboard updates instantly with detection alerts, visualizing animal type and event time.
  • The dashboard logs all detections for historical analysis and can trigger alert sounds or notifications.
  • Farmers monitor field animal activity conveniently through any network-connected device without needing internet.

System Benefits and Features

  • Edge AI Processing: Both modules process data locally, reducing latency and eliminating reliance on internet connectivity.
  • User-Friendly Dashboards: Flask web interfaces provide farmers real-time monitoring on simple devices.
  • Long-Range Communication: LoRa enables animal intrusion alerts from remote and wide farmlands.
  • Data Logging: Both systems save detection results in CSV format for trend analysis and reporting.
  • Low-Power Hardware: ESP32-CAM and LoRa modules are optimized for energy efficiency.
  • Scalability: More ESP32 units can be deployed across fields for distributed monitoring.
  • Real-Time Alerts: Farmers receive immediate notifications to take timely protective actions.

This integrated system offers a comprehensive solution for smart agricultural threat management by detecting crop threats and wild animal intrusions with edge computing, long-range communication, and intuitive web-based dashboards tailored for rural environments.
